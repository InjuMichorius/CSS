Form Validation using raw css

Valid

This example uses the :valid and :unvalid property. The disadvantage is that the input immediately indicates invalid when the user is typing. Only when the user has typed the correct answer it will be seen as valid.

Not focus

This example uses the :invalid:not(:focus) property. The invalid will not work when the input is focused, but instead be white. When the user clicks away from the input, the valid or invalid property will work.
